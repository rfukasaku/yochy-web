# yochy-web

## Setup
```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```

## deploy前の動作確認
```bash
# 静的ファイルをpublicディレクトリに出力
$ yarn generate

# サーバーを起動
$ yarn start
```

## Firebase Hostingにdeployする
```bash
# 静的ファイルをpublicディレクトリに出力
$ yarn generate

# デプロイ
$ firebase deploy
```
